# 国家大学生就业服务平台使用指南

`更新 26-02-05` \| `发布 26-02-01`

听说假期中可以关注该服务平台，找到招聘信息，一起看看怎么使用吧。如果觉得该平台有用，除了假期里关注，平时也可以多使用。网站链接：[国家大学生就业服务平台↗]。

初步使用该平台后，个人觉得还有挺有用的。推荐理由如下：

- 日常有众多岗位可查询到。除了平台提供的以外，还汇聚了近 20 个其他招聘平台的信息。
- 还有诸多网络线上招聘会的岗位。
- 国字头平台，岗位信息可信度有保障。

## 网站简介

通过 [网站简介↗]，先了解这个网站能干什么。原文摘要如下：

国家大学生就业服务平台是由教育部主管、教育部学生服务与素质发展中心运营的服务于高校毕业生及用人单位的公共就业服务平台。

- **精准推送职位**。根据往年招聘情况精准推送至毕业生，精准共享至各省、各校就业网，实现高精准、广覆盖。
- **线上线下结合**。采取手机版、二维码等移动互联网模式实现线上线下服务相结合。
- **大型网络招聘会**。定期举办全国、各省市联合招聘会、重点领域招聘会，构建人才招聘零距离通道。
- **生源信息查询**。分学历、专业查询全国800多万高校毕业生生源分布。
- **大数据支撑**。每年800多万就业数据、800多万生源数据、学信网1000万新增用户以及3500万在校生数据库、1.7亿高等教育学历信息库，为用人单位精准、高效招聘提供数据与用户支撑。
- **学籍学历验证**。求职者学籍学历信息经过验证，单位也可自主查询验证。
- **重点领域带动**（可查看 [重点领域](#重点领域) 做了解）。推动高校毕业生到重点领域单位就业，实现高校毕业生更高质量更充分就业。
- **专属客户和推广服务**。专属客服，提供最优解决方案；专属品牌及知名度推广服务。

### 投递简历

如想投递简历，该平台要求先注册登录。尝试是可以注册成功的。然后需要输入相关资料，比如个人信息、教育背景、求职期望、实习经历，等等。初步尝试后发现：**这些信息，可能都可以不填的，**就可以投递简历了。

平台也希望你在平台上写个简历。但简历的样例描述不是很好，可能被误导，请留意。后续给出分析建议（待续）。

**建议：要上传附件简历。** 附件简历就是已经写好的、pdf版本的简历。因为在简历投递时，该平台提示：

```markdown
简历投递确认
您正在投递 xxxx（用人单位） 的 xxxx（求助岗位）
您已上传附件简历，投递时会发给用人单位
```

## 联合招聘

可能是和其他招聘网站联合举办的，因此叫联合招聘。对于要求职的同学，就是网上招聘会而已。点击 [联合招聘↗] 可进入联合招聘页面。

> 岗位有标注“重点领域”/“精选”的，请参考 [重点领域](#重点领域) / [岗位精选计划](#selected-program) 知晓含义。

- **2026届高校毕业生全国网络联合招聘**（ 进入联合招聘页面后，该招聘会就在显要位置。后续位置可能会变化）

    ![2026届高校毕业生全国网络联合招聘](https://t2.chei.com.cn/student/ncss/cdn/job/img/4gem5mbkvm1z1v3o.png)

    可以看到页面有 `>>>点击此处进入联合招聘会主场<<<`。可以点击后进入该招聘会的主会场，然后搜索职位。如何搜索，和本文的 [职位信息](#职位信息) 类似，此处从略。
    
    联合招聘页面上关于该招聘会的协办单位的图标，大部分也是可以点击进入到分会场的。感兴趣同学可以进入分会场看看。‼️ 进入分会场的方式容易被忽略，或不容易进入，此处提醒下。

- **其他招聘会**

    联合招聘主页面下部，有全部、分行业就业大市场、分区域就业大市场、综合就业大市场，感兴趣同学可以进入相关招聘会看看机会。

- **岗位精选计划**<span id="selected-program"></span>

    （网页信息提示）岗位精选计划：即系统根据企业的信息完整度、用户活跃度、岗位发布情况、简历处理率进行评分计算，自动选取80分以上企业成为岗位精选企业，其岗位成为精选岗位。

    所以，精选企业（以及精选岗位），不一定是薪酬好、发展前景好等的企业/岗位，只是在“国家大学生就业服务平台”上比较活跃的企业。

    感兴趣同学可以看看精选计划。

## 重点领域

更多信息可查看 [重点领域↗] 页面。此处摘录 “重点领域推荐单位申请须知”（[原文链接↗](https://www.ncss.cn/student/keyunits/intropart.html)），供了解该平台的重点领域是指什么。从信息推测，重点领域是推荐产生的。摘录部分信息如下：

为鼓励和引导高校毕业生到国家重点领域就业，教育部学生服务与素质发展中心推出“重领域就业信息服务平台”，促进“双一流”高校与重点领域单位校企对接，现开展重点领域位推荐工作。

国家大学生就业服务平台及省级、校级就业网将为入选的重点领域推荐单位共享和精准送职位信息，并提供重点领域校企对接、网络招聘、重点推介宣传、生源信息查询等专免费服务。具体申请流程如下：

1. 申请单位填写《重点领域推荐单位申报表》（下载表格）；
2. 申请单位请推荐部门（省级就业指导中心或高校就业工作部门）签署推荐意见；
3. 将填写完毕的《重点领域推荐单位申报表》扫描发送至我中心联系人电子邮（wangym#chsi.com.cn 将#替换成@）。

申请推荐成功后，单位信息将纳入“重点领域就业信息服务平台”，获得“重点领域”专属识，并短信通知申请成功。

## 宏志助航

点击 [宏志助航↗] 可进入宏志助航网站，不再是该平台的一个板块。该网站主办单位也是教育部学生服务与素质发展中心。

该网站首页提示：宏志助航，全国高校毕业生就业能力培训网络平台；“宏志助航”网站是服务于教育部“宏志助航”帮扶计划，面向全国高校学生，提供就业能力课程学习和服务的公益性网站。以下对该网站的部分板块做简介。

**从各个板块看，宏志助航网站主要承载了一些课程。** 相关板块如下：

- 新闻动态
    
    发布了一些新闻。通过这些新闻，可以对宏志助航计划有进一步理解和了解。

- 专业类课程

    涉及近 20 个类别，比如：电子与信息类、农林牧渔类、土木建筑类、水利类、……。
    
    在电子与信息类中有一些课程，比如：计算机网络、WEB程序设计、华为5G/数通/数据库等认证课程、阿里淘宝运营等课程、集成电路等课程。

- 通用类课程
    
    有就业指导类、宏志案例类、就业直播课。前 2 类课程不是很多，有兴趣同学可以逐个浏览下。
    
    点击就业直播课就跳转到 [学职平台-就业指导课↗]。学职平台主办单位也是教育部学生服务与素质发展中心。

- 招聘信息
    
    点击招聘信息就回到 [国家大学生就业服务平台↗]。

- 创业服务
    
    点击创业服务，就跳转到 [全国大学生创业服务网↗]。该网站版权所有是教育部学生服务与素质发展中心。

- 职业测评
    
    点击职业测评就跳转到 [学职平台-职业测评↗]。
