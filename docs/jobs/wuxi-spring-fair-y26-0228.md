## 销售运营专员

- 单位名称：无锡中萃食品有限公司
- 职位名称：销售运营专员
- 招聘人数：2
- 需求专业：会计学、统计学、信息管理与信息系统、市场营销等相关专业。
- 学历要求：本科	

**职位信息**

1. 负责收集、整理和分析客户需求和市场信息，联系潜在客户并建立合作关系;
2. 建立良好的客户关系，对长期客户进行定期的跟踪，做好市场及客户的维护;
3. 对签约项目进行协调跟进，及时帮助客户解决售前、售中、售后可能出现的问题;
4. 根据市场发展和公司战略规划，根据产品特点和自身优势制定适合的营销计划并落实执行;明确目标市场，通过各种渠道和资源落实客户销售计划，并制定针对性的销售方案;
5. 协助部长收集和分析细分市场不同场景的各项信息数据，搜集整理客户反馈，开展消费者满意度调研。进行市场需求分析，对产品提出合理化建议，为营销决策提供参考依据；
6. 完成领导交办的其他工作。


**职位要求**

1. 本科及以上学历，会计学、统计学、信息管理与信息系统、市场营销等相关专业；
2. 具有敏锐的市场洞察力，对销售工作有较高的热情;
3. 具备良好的销售意识、沟通技巧和谈判技巧，普通话流利;
4. 具备较强的学习能力、良好的抗压能力和解决问题的能力，能较好的维护好客户关系;
5. 对于有相关专业知识或相关工作经验的可放宽学历专业或工作年限要求。

## 软件工程师（AI算法）

- 单位名称：无锡市环保集团有限公司
- 职位名称：研发部-软件工程师（AI算法）（无锡大禹科技有限公司）
- 招聘人数：1
- 需求专业：计算机类、计算机（软件、网络管理）类
- 学历要求：硕士	

**职位信息**

1、负责多源异构数据的清洗、去噪与特征工程，确保数据质量满足建模需求；
2、运用数据挖掘算法，对多维度数据进行深度分析，挖掘潜在信息；
3、结合业务场景设计特征分析方案，提取关键因子建立模型，通过数据的分析不断优化模型参数；
4、设计轻量级部署方案，支持模型在边缘设备或云平台运行；
5、领导交办的其他事项。

**职位要求**

1、精通Python及核心库（Pandas/NumPy）；
2、熟悉数据挖掘全流程算法、掌握时序建模与空间分析技术；
3、了解深度学习框架（PyTorch/TensorFlow）、熟练使用 AI 编程工具辅助开发，具备精准的 Prompt 提问技巧。
